# New Year 2026 Countdown

A terminal-based countdown to New Year 2026 with a fireworks animation.

## Description

This Python script displays a countdown to January 1, 2026. As the countdown runs, it launches colorful firework animations in the terminal. When the countdown reaches zero, it displays a "Happy New Year 2026!" message.

The animation is created using the `curses` library.

## Usage

To run the script, simply execute the following command in your terminal:

```bash
python main.py
```

**Note:** This script requires a terminal that supports the `curses` library. It is available by default on Linux and macOS. For Windows, you might need to install `windows-curses`.

Press `ESC` twice to exit the application.
